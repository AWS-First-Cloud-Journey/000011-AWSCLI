AWS Command Line Interface (AWS CLI) có hai phiên bản.

Để kiểm tra phiên bản đang cài đặt trên hệ thống của bạn, chạy lệnh ```aws --version```. Giá trị trả về là thông tin phiên bản mà bạn đã cài đặt.
- [AWS CLI phiên bản 1](1-ver-1/): AWS CLI phiên bản 1 là bản AWS CLI nguyên bản đầu tiên và vẫn sẽ được AWS hỗ trợ.
- [AWS CLI phiên bản 2](2-ver-2/): AWS CLI phiên bản 2 là bản AWS CLI mới nhất hỗ trợ tất cả các tính năng mới nhất của AWS. Một số tính năng có ở phiên bản 2 không được hỗ trợ tại phiên bản 1 và bạn cần phải nâng cấp phiên bản để sử dụng các tính năng đó.
